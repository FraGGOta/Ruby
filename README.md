# Ruby

## Install

```bash
gem install bundler
bundle install
```

## Run program

```bash
ruby ./lib/main.rb
```

## Run test

```bash
rspec
```

## Run rubocop

```bash
rubocop
rubocop -a # Auto-correct offenses
```
